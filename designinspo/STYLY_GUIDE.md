# FIRPLAK MarginOS – Design System (v1)

Goal: Modern, minimalist enterprise SaaS look. Premium but calm.

## Typography
- Use Google Font: Poppins
- Headings: font-semibold / tracking-tight
- Body: font-normal
- Keep line-height comfortable (leading-relaxed for paragraphs)

## Color
- Primary (dark blue): #0B1F3B
- Accent (blue): #1D4ED8
- Background: #F8FAFC
- Text primary: #0F172A
- Muted text: #64748B
- Borders: #E2E8F0

## Style
- Minimalist, fintech
- Large whitespace
- Soft shadows only (shadow-sm)
- Radius: rounded-2xl for cards/buttons
- No heavy gradients
- Use subtle blur/glass effect for decorative elements only:
  - backdrop-blur-md
  - bg-white/60

## Images
- When using background illustrations: apply blur and low opacity
- Do NOT use busy images; keep them abstract and soft
- Place images as decorative background layers, not content blockers

## Components
- Primary CTA button: solid dark blue
- Cards: border + soft shadow + hover lift (subtle)
- Icons: Lucide, stroke width consistent
